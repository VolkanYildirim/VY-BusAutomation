A desktop-based automation system developed in C# for managing bus ticket reservations and passenger tracking. This legacy project demonstrates fundamental object-oriented programming (OOP) principles and Windows Forms architecture from the early stages of my software engineering journey.

### ⚙️ Security & Privacy Philosophy
- **Zero Data Transmission:** This application operates entirely offline. No passenger data is transmitted to external servers, ensuring strict local privacy.
- **No Telemetry:** Free from any third-party tracking or background telemetry services.

### 🚀 Key Features
- Interactive graphical user interface for seat selection and passenger registration.
- Local state management for dynamic ticketing operations.

### 🔧 Build & Run
1. Clone the repository and open `C_Sharp_Form_Otobus_Otomasyonu.sln` using Visual Studio.
2. Set the build configuration to `Debug` or `Release` (Any CPU).
3. Build the solution (`Ctrl + Shift + B`) and start the application.
